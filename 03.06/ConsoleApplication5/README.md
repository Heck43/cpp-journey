# Каталог библиотеки — работа с файлами

> приложение для хранения книг: добавление, поиск и сохранение в файл

---

## Условие

Реализовать консольный каталог книг:
- класс `Book` с полями `title` (название) и `author` (автор)
- перегрузка `<<` и `>>` для записи/чтения из потока
- сохранение каталога в текстовый файл
- загрузка из файла при старте
- меню: добавить книгу, просмотреть список, найти по названию/автору, сохранить, загрузить

---

## Реализация

**Класс Book** использует геттеры и сеттеры для доступа к приватным полям.

**Перегрузка `<<`** записывает книгу в формате `Название|Автор\n` — разделитель `|` удобен для парсинга.

**Перегрузка `>>`** читает строку до `|` через `getline`, потом читает автора. Обрабатывает символ `\r` для совместимости с Windows.

**Хранение** — `vector<Book>`, все операции работают с ним.

**Поиск** использует `string::find()` — ищет подстроку и в названии, и в авторе.

## От себя

тут я первый раз серьёзно поработал с файлами в c++. `ifstream` / `ofstream` довольно простые, но с перегрузкой операторов ввода/вывода файловый код становится очень чистым.

особенно понравилось что `while (in >> book)` сам останавливается когда файл заканчивается. не надо проверять EOF вручную <3

символ `\r` в конце строки при чтении из файла — неожиданный сюрприз от Windows :3 пришлось добавить очистку
