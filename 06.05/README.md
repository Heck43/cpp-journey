## Задача 1 — Человек

Класс `Person` хранит ФИО через `char*` — три отдельных поля в куче.
Для даты рождения написан отдельный класс `Date(день, месяц, год)`.

**Конструктор копирования** обязателен, потому что у нас `char*`.
Без него копия и оригинал смотрели бы на одну и ту же память в куче —
при удалении одного объекта второй бы крашился:

```cpp
Person p3(p1);         // конструктор копирования
p3.setSurname("Сидоров"); // меняем только копию — оригинал не тронут
```

**Статический счётчик** — `static int count` объявлен в классе, но одно поле на всех:

```cpp
static int count;          // в классе — объявление
int Person::count = 0;     // вне класса — обязательное определение

Person::getCount()         // вызывается без объекта, через имя класса
```

Счётчик увеличивается в конструкторах и уменьшается в деструкторе.

## Задача 2 — String

Три конструктора для разных случаев создания строки:

```cpp
String s1;           // по умолчанию: буфер 80 символов, пустая строка
String s2(200);      // буфер заданного размера
String s3("Привет"); // из готовой строки — размер = длина + 1
String s4(s3);       // копирование
```

`strncpy_s` вместо `strcpy_s` — принимает ещё один параметр: сколько максимум скопировать.
Это защищает от выхода за границы буфера если строка длиннее.

`input()` использует `cin.getline(data, capacity)` — читает строку с пробелами целиком.

## Задача 3 — Array

Динамический массив `int*` с полным набором операций.

**`resize()`** — самое интересное место. Нельзя просто поменять размер `data` — память выделяется навсегда. Нужно:
1. создать новый массив нужного размера
2. скопировать старые данные
3. удалить старый массив
4. переключить указатель на новый

```cpp
int* newData = new int[newSize]();  // () — инициализация нулями
// копируем min(newSize, size) элементов
delete[] data;
data = newData;
size = newSize;
```

Конструктор копирования делает **глубокую копию** — копирует сами данные, а не указатель:

```cpp
Array arr2(arr);  // у arr2 своя память, изменения не влияют на arr
```

## Задача 5 — Телефонная книга

Два класса: `Contact` (один абонент) и `PhoneBook` (коллекция).

`fullName` — в куче через `char*`, остальные поля — `char[]` фиксированного размера.

`PhoneBook` хранит `vector<Contact*>` — вектор указателей.
В деструкторе удаляет каждый `Contact` через `delete`:

```cpp
~PhoneBook() {
    for (Contact* c : contacts) delete c;
}
```

**Поиск** через `strstr()` — ищет подстроку внутри строки:

```cpp
if (strstr(c->getFullName(), query) != nullptr)
    // подстрока найдена — выводим контакт
```

**Удаление** через итератор — `contacts.erase(it)` удаляет элемент из вектора по позиции.
