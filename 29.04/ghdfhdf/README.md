## Задача 1 — Товар на складе

Переписать процедурный код со `struct` и отдельными функциями в класс `Product`.

Класс хранит `char name[50]`, `int quantity`, `double price` в `public`.
Методы `print()` и `getTotalCost()` перенесли внутрь — теперь вызываются у объекта, а не принимают его параметром.

`strcpy_s(item.name, sizeof(item.name), "Монитор")` — для `char[]` обычное `=` не работает, нужна безопасная версия копирования строки.

## Задача 2 — Лампочка с выключателем

Класс `LightBulb` с полем `bool isOn` в `private`.

Снаружи нельзя написать `bulb.isOn = true` — только через методы:
- `turnOn()` — включить
- `turnOff()` — выключить  
- `toggle()` — переключить на противоположное (`isOn = !isOn`)
- `getState()` — вернуть `"ON"` или `"OFF"`

Конструктор `LightBulb() : isOn(false)` — при создании лампочка выключена.

## Задача 3 — Веб-ссылка

Класс `WebLink` с полями `char* url` и `char* description` — указатели на строки в куче.

В конструкторе выделяем память через `new char[strlen(...) + 1]` и копируем строку через `strcpy_s`. `+1` — для нулевого символа `\0` в конце.

Деструктор `~WebLink()` освобождает оба `delete[]` — без него была бы утечка памяти.

## Задача 4 — Банковский счёт

Класс `BankAccount` — все три поля (`owner`, `accountNumber`, `balance`) в `private`.

`deposit()` проверяет что сумма > 0. `withdraw()` проверяет что денег хватает — если нет, возвращает `false` и выводит ошибку. Конструктор проверяет что начальный баланс не отрицательный.

`bool withdraw(...)` — важно что возвращает именно `bool`, а не `void`, чтобы вызывающий код мог узнать получилось ли снять.

## Задача 5 — Термометр

Три конструктора с одним именем — C++ выбирает нужный по количеству аргументов:

```cpp
Thermometer t1;                    // по умолчанию: 0°C, "Неизвестно"
Thermometer t2(36.6);              // только температура, место = "Неизвестно"
Thermometer t3("Улица", -15.5);    // и место, и температура
```

Конструктор по умолчанию делегирует в конструктор с параметрами: `Thermometer() : Thermometer(0.0, "Неизвестно") {}` — не дублирует логику.

Перевод в Фаренгейты: `F = C * 9/5 + 32`.

## Задача 6 — Прямоугольник

Все методы — `inline` прямо в теле класса. `const` после метода = инспектор, только читает поля:

```cpp
double getArea()  const { return width * height; }
bool   isSquare() const { return width == height; }
```

Конструктор с тернарным оператором вместо if:

```cpp
width = (w > 0) ? w : 1.0;  // отрицательный размер → заменяем на 1.0
```
