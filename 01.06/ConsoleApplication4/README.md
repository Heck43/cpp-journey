# Множественное наследование — класс Car

> автомобиль как сумма своих частей: колёса + двигатель + двери

---

## Условие

Продемонстрировать множественное наследование:
- базовые классы: `Wheels` (колёса), `Engine` (двигатель), `Door` (двери)
- производный класс `Car` наследует сразу от всех трёх
- каждый базовый класс имеет свой метод `display()`
- `Car::display()` вызывает все методы сразу

---

## Реализация

Три базовых класса независимы друг от друга. Класс `Car` объявлен как:
```cpp
class Car : public Wheels, public Engine, public Door
```

Конструктор `Car` инициализирует все базовые классы через список инициализации. Метод `display()` в `Car` вызывает `Wheels::display()`, `Engine::display()` и `Door::display()`.

Также показано обращение к методам базовых классов напрямую через `myCar.Engine::display()`.
